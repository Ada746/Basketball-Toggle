# Basketball-Toggle
This project is an interactive toggle switch inspired by basketball.

When the toggle is clicked, a basketball jumps toward the hoop and drops into the net using CSS animations and JavaScript.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Interactive toggle switch
- Smooth CSS animations
- Basketball jump and drop effect
- Simple UI design

## How to Run
1. Download the project
2. Open `index.html` in your browser
